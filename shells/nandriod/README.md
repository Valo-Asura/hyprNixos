# Nandriod Shell

This profile launches the read-only reference shell from:

```text
/home/asura/Downloads/test/asura-quickshell-main/shell.qml
```

Use:

```bash
switch-shell nandriod
switch-shell vibeshell
```

The reference tree is not modified by this setup. Runtime logs are written to
`~/.local/state/asura-shell/nandriod.log`.
